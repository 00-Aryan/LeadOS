# Done Criteria

A task is done only when all applicable criteria are satisfied:

- Scope is implemented.
- Tests are added or updated where needed.
- Format passes.
- Lint passes.
- Tests pass.
- Docs are updated if behavior changed.
- Manual work is listed.
- Risks are listed.
- No unrelated changes were made.

Do not mark a task done when validation fails. If validation cannot run, report the blocker and the exact manual validation required.
