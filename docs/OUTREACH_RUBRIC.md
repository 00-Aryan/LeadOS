# Outreach Rubric

## Objective

The outreach service prepares drafts only. It must not send messages automatically.

## Supported MVP channels

```text
short_email
whatsapp_message
```

## Required input

```text
business_name
category
location
audit_findings
score_reason
offer_angle
channel
```

## Required output

```text
subject_line
message_body
personalization_notes
assumptions_used
review_required
```

## Quality rules

A draft must be:

- Specific to the business
- Based only on known facts
- Short enough for the selected channel
- Clear about the next step
- Respectful and non-manipulative

## Forbidden behavior

The draft must not:

- Invent revenue numbers
- Claim the business is losing customers without evidence
- Claim the sender has audited private data
- Use fake familiarity
- Create urgency through fear tactics
- Promise guaranteed results

## Manual review

Every outreach draft requires evaluation before use. Low-scoring or high-risk drafts must be revised before a human sends them.
