# Engineering Rules

## Product focus

LeadOS is a lead intelligence product.

The product loop is:

- CSV import
- audit
- scoring
- outreach draft
- evaluation

## Council rule

Before each task, define:

- goal
- constraints
- owner roles
- acceptance criteria
- tests
- review gate

## Coding rule

Prefer small typed services. Return structured errors. Avoid hidden side effects.

## Current deferred work

Do not add scraping, CRM sync, ProjectOS sync, content tool sync, billing, or deployment yet.
