# Drift Control

## Product idea

LeadOS is a lead intelligence product. It is not a generic automation platform.

## Current loop

- import leads
- audit digital presence
- score opportunity
- draft outreach
- evaluate draft

## Drift checks

Before adding a feature, ask:

- Does it improve the current loop?
- Can it be tested independently?
- Does it avoid hidden side effects?
- Does it keep future integration optional?

If the answer is no, defer it.
